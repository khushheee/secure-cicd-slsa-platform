# Secure CI/CD Pipeline with SLSA Provenance

An automated, security-first CI/CD pipeline built with GitHub Actions for building, scanning, and deploying Docker images with full supply chain traceability.

## Key Features
- **Automated CI/CD**: Build and push Docker images to DockerHub on every push to main
- **DevSecOps**: Integrated security scans detect secrets and vulnerabilities before deployment
- **SLSA Level 3**: Generates cryptographically signed SLSA provenance attestations for supply chain security
- **Containerized**: Fully dockerized application ready for kubernetes deployment

## Tech Stack
GitHub Actions, Docker, DockerHub, SLSA, YAML
